## 🚀 How to Use This Package

### For **Elementor Pro** Users

1. **Install WordPress** + Elementor Pro
2. **Copy CSS** from `/css/variables.css` into Customizer > Additional CSS
3. **Create new page** → Edit with Elementor
4. **Add HTML widget** → Copy content from `/templates/home.html`
5. **Style with classes** from `components.css`
6. **Repeat** for other pages

**Time:** 4-6 hours

---

### For **Bricks Builder** Users

1. **Install WordPress** + Bricks Builder
2. **Import design tokens** from `/css/variables.css`
3. **Build pages** using component library
4. **Reference** HTML structure from `/templates/`

**Time:** 5-7 hours

---

### For **Custom Theme** Developers

1. **Create child theme** (or custom theme)
2. **Enqueue stylesheets** from `/css/` folder
3. **Convert templates** to PHP page templates
4. **Integrate** navigation menus and footer

**Time:** 8-12 hours
